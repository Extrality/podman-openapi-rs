# Podman OpenAPI Client

Auto-generated Rust client for the Podman API.

Refer to the [Podman API documentation](https://docs.podman.io/en/latest/_static/api.html) for additional information.

## Usage

```toml
[dependencies]
podman-openapi = { git = "https://github.com/Extrality/podman-openapi-rs.git", rev = "71647286f6c88d8322e0c00f7b6dc929985eefda" }
```
